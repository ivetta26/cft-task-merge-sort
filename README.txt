Версия java - 16
Система сборки - Gradle 7.4
Сторонние библиотеки - launch4j для сборки проекта в формате exe

Чтобы запустить проект
1) его надо скопилировать clean build
2) вызвать команду launch4j для создания exe файла
3) запустить через .\cft-task-merge-sort.exe -i -a out.txt in1.txt in2.txt in3.txt

Собранная и работающая версия проекта уже есть в папке working_version
